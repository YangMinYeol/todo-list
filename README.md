## Todo List

Next.js를 공부하며 직접 만든 할 일 관리 서비스입니다. 할 일을 적는 것에 그치지 않고, 사진과 메모를 남겨 더 꼼꼼하게 기록할 수 있도록 제작했습니다.

### 🔗 배포 링크

## 🔗 배포 링크
- **URL**: [https://todo-list-theta-pink-27.vercel.app/](https://todo-list-theta-pink-27.vercel.app/)

### 📖 프로젝트 설명

#### 어떤 프로젝트인가요?

오늘 할 일을 등록하고, 완료한 일은 체크해서 따로 모아볼 수 있는 대시보드입니다.
각 할 일마다 상세 페이지가 있어서 사진을 올리거나 더 자세한 메모를 적을 수 있습니다.

#### 만들면서 신경 쓴 점들

- 한눈에 들어오는 목록: 해야 할 일(TO DO)과 다 한 일(DONE)을 자동으로 나눠서 보여주게 만들었습니다.

- 불필요한 클릭 방지: 상세 페이지에서 아무것도 수정하지 않았을 때는 '수정 완료' 버튼이 활성화되지 않게 해서 불필요한 저장을 막았습니다.

- 이미지 업로드: 이미지 이름에 특수문자가 있거나 용량이 너무 크면(5MB 이상) 미리 알려주도록 처리해서 에러를 방지했습니다.

- 로고 새로고침: 어디서든 상단 로고만 누르면 메인 화면으로 돌아오면서 최신 목록을 다시 불러오게 설정했습니다.

#### 사용한 도구들

프레임워크: Next.js (App Router)
언어: TypeScript
스타일: Tailwind CSS
데이터 통신: Axios

#### 사용 방법

1. 패키지 설치
   터미널을 열고 필요한 도구들을 먼저 설치합니다.

```
npm install
```

2. 서버 실행하기
   설치가 끝났다면 아래 명령어로 앱을 실행할 수 있습니다.

```
npm run dev
```

3. http://localhost:3000을 입력
